# Defensive Security Intro (TryHackMe)

## What is Defensive Security

Defensive Security focuses on protecting systems, networks, and data from cyber attacks.

Also known as:

Blue Team

---

## Defensive Security Roles

SOC Analyst
- monitors networks
- detects suspicious activity

Incident Responder
- investigates attacks
- contains threats

Security Engineer
- builds security systems
- maintains security tools

Digital Forensics
- analyzes evidence after an attack
- investigates how the attack happened

---

## Defensive Security Tools

IDS (Intrusion Detection System)
- detects suspicious activity on the network

Firewalls
- control what traffic enters the network

Security Policies
- rules for safe system usage

SIEM (Security Information and Event Management)
- central system that collects logs and security events

---

## SOC (Security Operations Center)

SOC is a team that monitors and protects an organization’s IT infrastructure.

Tasks include:

- reviewing alerts
- investigating anomalies
- responding to incidents
