## LAN Topologies

### What is LAN
LAN = Local Area Network  
A LAN connects multiple devices within a limited area such as a home, office, or school network.

---

### Star Topology
Devices connect to a central device (usually a switch).

Pros:
- Reliable
- Easy to add new devices
- Easy to manage

Cons:
- If the central device fails, the entire network goes down
- More expensive due to extra cabling and hardware

---

### Bus Topology
All devices share a single backbone cable.

Pros:
- Cheap to set up
- Requires less cabling

Cons:
- Bottlenecks easily
- Hard to troubleshoot
- Single point of failure (if the cable breaks, the network fails)

---

### Ring Topology
Devices are connected in a circular loop.

Pros:
- Less network congestion
- Data flows in a structured path

Cons:
- If one device or cable fails, the entire network can break
- Data may need to travel through multiple devices before reaching the target

---

### Switch
A switch connects multiple devices inside a LAN.

Key points:
- Works at Layer 2 (MAC addresses)
- Sends packets only to the correct device
- Reduces unnecessary network traffic

---

### Router
Routers connect different networks together.

Example:
- Home network ↔ Internet

Key concept:
Routing = the process of directing data between networks.
