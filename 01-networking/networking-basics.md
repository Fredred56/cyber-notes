## TryHackMe - What is Networking

### Network
A network is a group of connected devices that communicate with each other.

### Internet
The Internet is a global network made of many smaller networks.

Invented by:
Tim Berners-Lee (World Wide Web)

### IP Address
Identifies a device on a network.

Example:

192.168.1.1

Each section is called an **octet**.

### MAC Address
Unique hardware address for a network interface.

Example:

a4:c3:f0:85:ac:2d

### Ping
Used to test connectivity between devices.

Uses protocol:

ICMP

Example command:

ping 192.168.1.254
