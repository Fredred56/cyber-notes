# Linux Basics (RHEL / General)

## Core concepts
- Linux is a multi-user OS
- Everything is a file (mostly)
- Permissions control access (user / group / others)
## Navigation & files
```bash
pwd
ls -la
cd /etc
cd ~
mkdir test
touch file.txt
cp file.txt copy.txt
mv copy.txt moved.txt
rm moved.txt
```
## Help commands
```bash
man ls
ls --help
info coreutils
```
## Useful notes
- `.` = current directory
- `..` = parent directory
- `~` = home directory
- `Ctrl + C` stops a running process
